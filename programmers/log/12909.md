## Solution 1

```java
import java.util.Deque;
import java.util.LinkedList;

class Solution {
    boolean solution(String s) {
        Deque<Character> stack = new LinkedList<>();
        for (char ch : s.toCharArray()) {
            if (ch == '(') {
                stack.offerLast(ch);
            } else {
                if (stack.isEmpty()) {
                    return false;
                }
                stack.pollLast();
            }
        }

        return stack.isEmpty();
    }
}
```
