## Wrong 1

```java
import java.util.Arrays;

class Solution {
    public int solution(int[] citations) {
        Arrays.sort(citations);
        int answer = 0;
        for (int i = 0; i < citations.length; i++) {
            int h = citations[i];
            int citedPapers = citations.length - i;

            if (citedPapers >= h) {
                answer = h;
            } else {
                break;
            }
        }
        return answer;
    }
}
```
