# [문자열 내 p와 y의 개수](https://school.programmers.co.kr/learn/courses/30/lessons/12916)

> `프로그래멋 코딩테스트 문제 풀이 전략 137pg`

## Solution 1

```java
class Solution {
    boolean solution(String s) {
        s = s.toLowerCase();
        int countP = 0;
        int countY = 0;

        for (char c : s.toCharArray()) {
            if ('p' == c) {
                countP++;
            } else if ('y' == c) {
                countY++;
            }
        }
        
        return countP == countY;
    }
}
```
