## 정리

- 숫자 형변환 주의

## My Solution 1

```java
import java.util.*;

class Solution {
    public int[] solution(long n) {
        List<Integer> result = new ArrayList<>();
        while (n > 0) {
            result.add((int) (n % 10));
            n /= 10;
        }

        return result.stream().mapToInt(Integer::intValue).toArray();
    }
}
```

### Wrong 1 - 1

```java
class Solution {
    public int[] solution(long n) {
        List<Integer> result = new ArrayList<>();
        while (n > 0) {
            // 아래와 같이 되면 n % 10 의 결과가 정수형이 되는 것이 아니라
            // n 이 정수형이 되버림
            result.add((int) n % 10);
            n /= 10;
        }

        return result.stream().mapToInt(Integer::intValue).toArray();
    }
}
```
