# 이진 탐색

## 범위

### 특정 범위 가장 앞 인덱스

```java
public static int lowerBound(int[] arr, int target, int start, int end) {
    while (start < end) {
        int mid = (start + end) / 2;
        if (arr[mid] >= target) {
            end = mid;
        } else {
            start = mid + 1;
        }
    }
    return end;
}
```

### 특정 범위 가장 끝 인덱스

public static int upperBound(int[] arr, int target, int start, int end) {
